## Public static void main(String[] args)
- Javaの実行に必須のメソッド
## System.out.println()
- 引数の出力
## ;
- 命令の区切り
## /*  */
- Javaにおけるコメント
- 複数行の場合の書き方
## //
- 1行の場合は//から行末までがコメント
- ショートカットキーはctrl + /
## int ○○ = 150;
- ○○に整数値を紐づける
- ○○に150を代入
- ○○に再代入をする場合intは不要
## double ○○ = 1.1
- ○○が小数点を含む変数の場合
## final double ○○ = 1.1
- 変数○○の再代入を禁止
- 値を再代入できない変数は「定数」。○○を大文字で記述
## String ○○ = ken
- 変数○○が文字列
